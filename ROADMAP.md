# Solo Leveling App — фичи и план

## Концепция
Фитнес-приложение в стиле «Поднятие уровня в одиночку». Telegram мини-апп.
Для себя + друзья/клуб. Геймификация: уровень, EXP, ранги E→S, ежедневные квесты.

## Юридическое (важно)
- Кадры из оригинальной манги/аниме Solo Leveling в продукт НЕ вставляем (копирайт, риск бана мини-аппа).
- Делаем СВОЙ арт в той же эстетике (стиль не охраняется): порталы, силуэт охотника, окна Системы.
- Фишка: загрузка своего фото → AI-аватар «ты в стиле охотника» (image-to-image, MuAPI). Законно.

## Бэкенд / логика (TODO)
### Фото → калории (ЭТАП 1 — приоритет, ТЗ от Любы)
- Пользователь: фоткает еду ИЛИ этикетку + пишет комментарий (что съел, примерно на глаз / состав с этикетки)
- Фото + текст → AI-модель со зрением (GPT vision / Claude vision) → распознаёт блюдо, оценивает ккал + БЖУ
- Точность приблизительная (±20%) — это ок, цель «видеть как ешь», а не аптечная точность
- Результат → запись в дневник, обновление верхней таблицы БЖУ/ккал
- Показывать ДЕФИЦИТ/ПРОФИЦИТ относительно нормы (цель: похудеть=дефицит, набрать=профицит)
- Прототип: имитация по словарю foodDB (analyze/addMeal/updBalance) — работает визуально
- НЕ делаем пока: планирование меню, «что приготовить» — слишком сложно, потом

### Вода/напитки (как Water Time, реализовано в прототипе)
- Выбор напитка (вода/чай/кофе/сок) + объём (200/250/500) + тап «добавить»
- Коэффициент усвоения (hydro): вода 1.0, чай 0.9, кофе/сок 0.8
- Норма мл считается от веса; прогресс-бар; напоминания раз в 2 ч

### Уведомления (бот, как Duolingo)
- Не заходил весь день → вечерний пинок + обратный отсчёт до штрафа
- «Тень напоминает» (мягко) → строгий тон ближе к дедлайну (аналог злого/доброго Дуолинго)
- Друг отстаёт → напоминание тебе; кнопка «Пнуть» → уведомление другу
- Достижение друга → «X поднялся до ранга B»
- Реализация: cron-проверки (n8n / бот), рассылка через Telegram Bot API
