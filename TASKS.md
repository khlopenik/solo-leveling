## ✅ СДЕЛАНО (в прототипе, дизайн)
- Онбординг «Создание игрока»: имя, дата рождения, пол, рост, вес, цель, активность → расчёт норм
- Необязательный блок фото тела в онбординге (3-4 ракурса, можно пропустить → базовая фигурка)
- Экран Система: окно квеста в стиле System (! + название + ✕, [Получено задание], ЦЕЛИ, [0/5]→[5/5], ВНИМАНИЕ)
- Статы на русском 2×2: 💪Сила / ⚡Ловкость / ❤️Здоровье / 🧠Интеллект
- Уровень, EXP, ранги E→D→C→B→A→S (авто по уровню), таймер дня, штраф
- Квест чтения (5 страниц → Интеллект)
- Питание: кольцо калорий, БЖУ с нормами и цветом, план vs факт по времени
- Фото→калории (этап 1, имитация): фото+коммент → ккал/БЖУ → дневник, дефицит/профицит
- Вода/напитки (как Water Time): тип (вода/чай/кофе/сок) + объём + коэф. усвоения + тап
- Прогресс: анализ состава тела (полный список метрик со статусами), 3 способа ввода
- Замеры сантиметром (11 параметров + динамика)
- График веса интерактивный: период (Неделя/Месяц/6мес/Год) + метрика (Вес/ИМТ/Жир%), статистика
- Фигурка тела (муж/жен по полу)
- Серия дней, достижения (сетка)
- Рейтинг (бывш. Гильдия): общий рейтинг top-5 + твоя позиция + весь топ
- Клан (сообщество), Пати (бади 1-на-1), Друзья, Лента достижений
- Профиль игрока по тапу (написать/в друзья/подписаться)
- Настройки приватности (5 переключателей публичности)
- Встроенный чат (пузыри, отправка, автоответ) + вкладка «Чаты» (личные/группы/пати)
- Переименование Охотник → Игрок
- Конструктор ежедневного квеста: игрок сам выбирает упражнения из 4 сфер + свои (кнопка ⚙)
- Серия дней — визуальные «денёчки» (закрашенные 🔥 + сегодня + будущие); достижения под серией
- Шрифт заменён Orbitron → Exo 2 (под референс QUEST INFO), убран лишний неон
- Питание: «план питания» убран → простой Дневник (что съел + время)

## ⚠ РЕЙТИНГ — алгоритм (продумать на бэке)
- Учитывать ОБЪЁМ квеста: у кого 6 упражнений в день, тот при равном усердии выше, чем у кого 2
- XP за день масштабируется от числа и сложности выбранных целей (дневной потолок)
- Чтобы догнать игрока с большим квестом — нужно делать пропорционально больше
- 4 сферы (Сила/Ловкость/Интеллект/Здоровье): минимум по 1 из каждой; доп. упражнения = бонус
